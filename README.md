 # Real Estate Website

## Overview

***A real estate website task from depi***


This is a Fullstack project for a real estate website built to help sellers to list their properties on the website and update and delete it if wanted, and user can search and browse various type of properties.
The website provides a user-friendly interface for browsing and searching for properties also for listting , update and delete properties.


## Technologies Used

### Front-End : 
  - React js
  - Bootstrap  / React-Bootstrap
  - CSS5
  - Swiper js
  - Axios
  - Redux and Redux toolkit 

### Backend : 
  - Node js
  - Mongoose
  - Express js
  - TypeScript

## Installation
To run this project locally, follow these steps (You need to install node js first):

1. **Open cmd**

2. **Clone the repository**:
    ```sh
    git clone https://github.com/MazenMarei/real-estate-website.git
    ```

3. **Navigate to the project directory**:
    ```sh
    cd real-estate-website
    ```
4. **Switch To fullstack branch**:
  ```sh 
  git branch fullstack
  ```

5. **Navigate to the Front end directory**:
```sh 
  cd client
```

6. **Install React and dependencies**:
```sh
npm i
```

7. **Navigate to the Back end directory**: 
```sh
cd ../ && cd server 
```

8. **Install dependencies**
```sh 
npm i && npm i typescript ts-node
```

9. **Navigate to the project directory**:
```sh
cd ../
```

10. **Finally you can open Start-Site.bat file to start the Frontend and Backend**
